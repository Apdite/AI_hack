# AI_hack

## Установка
1. Клонируйте репозиторий: `git clone https://github.com/Apdite/AI_hack.git`
2. Установите зависимости: `pip install -r requirements.txt`

## Структура проекта
- **models/**: Скрипты для моделей (YOLO, MMDetection).
- **ui/**: Интерфейсы на Gradio и Streamlit.
- **tests/**: Тесты.
- **data/**: Датасеты.
- **configs/**: Конфигурации.
